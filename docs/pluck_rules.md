# Pluck – Official Game Rules

## Deck
- 51 cards total
- All 2s removed except 2♣
- Big Joker
- Little Joker

## Opening Rule
- Player holding 2♣ must lead first trick
- 2♣ is lowest club
- If Clubs are trump, 2♣ is lowest trump

## Trump
- Dealer selects one suit as trump
- No "No Trump" option
- Big Joker > Little Joker > A > K > ... > 3 (of trump)

## Trump Opens
- Trump opens the moment any trump card is played
- Cannot lead trump until trump is opened
- Exception: If only trump cards remain in hand

## Following Suit
- Must follow suit if possible
- Jokers count as trump suit for following purposes
- If holding only a Joker and trump is led, Joker must be played

## Trick Resolution
- If any trump played → highest trump wins
- If no trump played → highest card of lead suit wins
- Winner leads next trick

## Hand Structure
- 17 cards dealt to each player
- 17 tricks per hand

## Pluck Phase
(To be implemented in engine)

- Plucks determined by quota vs tricks won
- Least plucks pluck first
- Must offer lowest card of selected suit
- Plucker must return higher card of same suit
- No duplicate suit plucks from same player
- Jokers cannot be forced in pluck
